# Random-Password-Generator-
A beginner friendly Python program that generates a secure, random password using letters, numbers, and symbols based on user input.

This is a simple Python project that generates a random and secure password based on user preferences.
The user can choose how many letters, numbers, and symbols they want, and the program creates a shuffled password accordingly.

Features
Takes user input for password length
Uses uppercase & lowercase letters
Includes numbers and special symbols
Randomly shuffles characters for better security
Beginner-friendly and easy to understand
Technologies Used
Python
random module

How to Run
Make sure Python is installed on your system
Clone this repository or download the file
Run the script:
python password_generator.py
Follow the prompts in the terminal
